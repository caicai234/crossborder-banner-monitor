## Banner 监控系统
- 老板要求监控7个跨境平台首页banner变化并推送提醒：Amazon/Shein/HomeDepot/Temu/AliExpress/Wayfair/Lowes
- 创建 banner-monitor.html 单文件应用
- 核心技术：
  - 通过 thum.io API 获取网站截图（width/800/crop/600，截取banner区域）
  - 多层CORS fallback：直接fetch → allorigins代理 → corsproxy.io
  - 感知哈希（pHash）：16x16灰度 → 256位哈希 → 汉明距离对比
  - 变化阈值可调（高10/中18/低30）
  - blob→objectURL→Image→canvas 避免canvas污染
- 功能：
  - 监控卡片网格：每个网站显示截图、状态灯、检测时间、变化次数
  - 状态：正常(绿)/已变化(红闪烁)/检测中(蓝)/错误(灰)/已建基线/未检测
  - 通知系统：浏览器Notification + Toast弹窗 + Web Audio提示音
  - 变化历史时间线（最多30条）
  - 截图对比弹窗：左右并排展示新旧截图
  - 监控控制：启动/暂停、检测间隔(5-60分钟)、灵敏度、手动检测
  - 倒计时显示下次检测时间
  - localStorage持久化（哈希、缩略图、历史记录）
- 部署到 CloudStudio：https://392fc13b90b347aba802e8180f8563f5.app.codebuddy.work/banner-monitor.html
- 注意：thum.io截图加载较慢(5-15秒)，串行检测每个间隔2.5秒
- 检测频率调整：默认改为1天(1440min)，选项为1h/6h/12h/1天/2天/7天
- Wayfair 画面问题：Wayfair 有严格反爬保护，thum.io返回空白图(11KB)、Playwright也被拦截("Access Denied")、所有CORS代理均失败
